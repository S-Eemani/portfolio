WHATSAPP CHATBOT IN PaaS ENVIRONMENT (VIA HEROKU)

A chatbot is a software application that is able to conduct a conversation with a human user through written or spoken language. While there are very complex chatbots out there, our chatbot is currently simple, and can display quotes, jokes or cat images (the functions it can do can be increased and personalised more) upon asking it to do so.

In our project, we have used ​Heroku​, which is a container-based cloud Platform as a Service (PaaS), to help host our chatbot. We have also used ​Twilio​ (a cloud communications platform) for the whatsapp interface. Apart from this, we also used a Jokes API, a cat images API, and a quote API (details of all these modules explained below). The code for our chatbot is written mainly in Python. We also used the Flask framework.

we had to configure the Twilio Sandbox. We used Twilio’s ‘Programmable Messaging’ option to set up a sandbox with our account. The whatsapp number is common to all, but it has a code with the word ‘join’ followed by another code, unique to the account we use to make it. All the users of the chatbot would have to connect via python code
